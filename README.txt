Задание Calculator
Написать калькулятор для простейших арифметических выражений.
Требования по оформлению
Приложение должно быть написано на языке Java без использования дополнительных  библиотек.
Готовое приложение упаковать в jar-файл calculator.jar, исходники в calculator-src.jar
Main class : edu.tsystems.test.calc.CalculatorStarter
Во время запуска в приложение передаются два параметра: 
1 – путь к входному файлу, каждая строка которого является арифметическим выражением.
Арифметическое выражение состоит из следующих символов: цифры, круглые скобки и знаки оперций “+”, ”-“, ”*”, “/” Приоритет операций как обычно.
Пример
(1+38)*4-5
7*6/2+8
-12)1//(
2 – путь к файлу вывода, в который  по завершению программы должны быть записаны результаты вычисления выражений из файла 1.
Ответы должны быть записаны построчно. 
В случае, если арифметическое выражение является некоректным, вывести в соответствующей строке слово ERROR. 
Пример
151
29
ERROR
